# Event (Event)

## What is event?
Google's event experience simplifies finding and attending events via Google Search and other Google products, such as Google Maps. This feature offers several advantages:

- More interactive results: Your events can appear in Google's event experience, showcasing your logo, event description, and more.
- Increased discovery and conversion: Users can engage with your event posting in a new way, increasing the likelihood of visits to your site. For example, Eventbrite experienced a 100% increase in typical year-over-year growth in traffic from Google Search.

Kindly Refer [Google Doc](https://developers.google.com/search/docs/appearance/structured-data/event) to know more about event rich result.

## What are necessary data for event rich result?
### 1. Event name
```html copy filename="event.html" /1-name/ {2}
<div>
  <p class="rjs-event-1-name">The Age of AI</p>
</div>
```

### 2. Mode

<Callout>Data attribute item</Callout>

Three Modes are,
- `online` - Online Only Event
- `offline` - Offline Only Event
- `mixed` - Hybrid or Online & Offline - Mixed Event.
```html copy filename="event.html" {3,5,7} /1-mode/
<div>
  
  <p>Attendance Mode: <span class="rjs-event-1-mode" data-mode="online">Online</span></p>

  <p>Attendance Mode: <span class="rjs-event-1-mode" data-mode="offline">Offline</span></p>
  
  <p>Attendance Mode: <span class="rjs-event-1-mode" data-mode="mixed">Mixed</span></p>

</div>
```

### 3. Start date
```html copy filename="event.html" {2} /1-from/
<div>
  <p>Starting Date: <span class="rjs-event-1-from">2024-02-26 05:23 PM</span></p>
</div>
```

### 4. End Date
```html copy filename="event.html" {2} /1-end/
<div>
  <p>Ending Date: <span class="rjs-event-1-end">2024-03-26 05:23 PM</span></p>
</div>
```

### 5. Status
Status data values are:
- `postponed` - Event Postponed
- `toonline` - Event Moved to Online from Offline
- `cancelled` - Event Cancelled
- `rescheduled` - Event Re-Scheduled
- `scheduled` - Event as per Scheduled

```html copy filename="event.html" /1-sts/ {2} /data-sts="scheduled"/
<div>
  <p>Event Status: <span class="rjs-event-1-sts" data-sts="scheduled">Scheduled</span></p>
</div>
```

### 6. Images - Thumbnails

<div>

### 9. Performer(s)
```html copy filename="event.html" /1-aname/
<div>
  
  <!--any number of performer possible-->
  <p>Actor/Performer:
    <span class="rjs-event-1-aname">DEAN</span>,
    <span class="rjs-event-1-aname">SAM</span>
  </p>

</div>
```

### 10. Organiser(s)
- `pnameP` - Person type Organiser
- `pnameO` - Organisation type Organiser

```html copy filename="event.html" /pnameO/ /pnameP/ {3,7}
<div>

  <p>Organised By <span class="rjs-event-1-pnameP">JOHN</span></p>

  <!--With link OR without link-->

  <p>Organised By <a href="https://www.organisationpage.com" class="rjs-event-1-pnameO">Cresteem</a></p>

</div>
```

### 11. Price & Currency

<Callout>Pass data as free only if event entry is free otherwise pass currency alpha-3 code</Callout>

```html copy filename="event.html" /1-cost/ /data-currency="free"/ /data-currency="inr"/ {4}
<div>
  
  <p>Event Fee: 
    <span class="rjs-event-1-cost" data-currency="free">FREE</span>
  </p>

  <!-- OR -->

  <p>Event Fee:
    <span class="rjs-event-1-cost" data-currency="inr">200 INR</span>
  </p>

</div>
```

### 12. Booking (Reservation) link
```html copy filename="event.html" /1-rlink/ {3}
<div>

  <a class="rjs-event-1-rlink" href="https://www.rjs-eventsite.com/book-event">Book Here</a>

</div>
```

## Function and parameters
    
    <Callout type='default'>By default input file is overwriiten with result so `destination` is optional</Callout>

    <Callout type='info'>This is only for API method.
    We prefer CLI method for keeping it simpler (refer - [Working with API & CLI](/how-to-use)).</Callout>

    ```js copy filename="func_params.js" {5}
    const richResultType = 'event';
    const filePath = 'event.html';
    const destination = 'dist/event.html'; /* optional */
    
    richie(richResultType, filePath, destination);
    ```
