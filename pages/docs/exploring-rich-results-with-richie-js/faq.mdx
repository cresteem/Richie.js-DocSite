# FAQ (FAQPage, Question, Answer)

## What is FAQ?

An FAQ page includes a series of questions and answers related to a specific topic. When correctly formatted, these pages might qualify for a rich result in Search and an Action on Google Assistant, enhancing your site's visibility and helping it connect with the appropriate audience.

Kindly Refer [Google Doc](https://developers.google.com/search/docs/appearance/structured-data/faqpage) to know more about FAQ rich result.

## What are necessary data for FAQ rich result?

### 1. FAQ Wrapper

```html copy filename="faq.html" {1,11} /rjs-faq/
<details class="rjs-faq">
  <summary class="question">
    <!--removed for brevity -->
  </summary>

  <p class="answer">
    <!--removed for brevity -->
  </p>
</details>
```

### 2. Question

```html copy filename="faq.html" /question/ {3-6}  /rjs-faq/
<details class="rjs-faq">
  <summary class="question">
    Can you provide a content management system (CMS) that allows me to update
    the website easily?
  </summary>

  ........<!-- reduced for brevity -->
</details>
```

### 3. Answer

```html copy filename="faq.html" /answer/ {5-8}  /rjs-faq/
<details class="rjs-faq">
  ......<!-- reduced for brevity -->

  <p class="answer">
    Certainly! We take pride in crafting custom
    <b>Content Management Systems (CMS) tailored</b> exclusively to your
    requirements.
  </p>
</details>
```

## Function and parameters

    <Callout type='default'>By default input file is overwriiten with result so `destination` is optional</Callout>

    <Callout type='info'>This is only for API method.
    We prefer CLI method for keeping it simpler (refer - [Working with API & CLI](/working-with-richie-js-api-and-cli)).</Callout>

    ```js copy filename="func_params.js" {5}
    const richResultType = 'faq';
    const filePath = 'faq.html';
    const destination = 'dist/faq.html'; /* optional */

    richie([richResultType], filePath, destination);
    ```
