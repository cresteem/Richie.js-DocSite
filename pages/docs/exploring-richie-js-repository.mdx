# Demonstration

You can find fabricated HTML and configuration files for **demonstration** in this [GitHub repository](https://github.com/cresteem/Richie-JS-Demo). Clone this repository to your local machine to **experiment** with configurations and view the HTML to understand how Richie JS works. This hands-on approach will help you better grasp the functionalities and integration of **Richie JS**.

## Clone demonstration repository

```bash filename="Terminal @Demo" copy
git clone https://github.com/cresteem/Richie-JS-Demo.git
```
